# Contributing to SecurityInsight

Pull requests are welcome here. The maintainer will review on this repo and
then bridge the change into the upstream monorepo monorepo, after which the
next release republishes the change here. Your commit attribution is preserved.

Bug reports and feature requests can also be opened as GitHub issues.

If you'd rather work with the full source tree (all solutions, all platform
modules), the upstream is at https://github.com/KnudsenMorten/AutomateIT.
